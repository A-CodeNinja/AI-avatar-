## 二、参考图片准备

### 🎨 参考图片用途

参考图片用于：
1. 训练AI理解民族风格特征
2. 作为风格参考库
3. 展示给用户预览

### 📐 图片规范

| 项目 | 要求 |
|------|------|
| 格式 | JPG/PNG |
| 尺寸 | 512×512px 或 1024×1024px |
| 大小 | < 500KB/张 |
| 内容 | 清晰、正面、光线均匀 |

每种民族需要包含：

- 艾德莱斯绸裙装

**图案元素**:

### 🌐 参考图片获取渠道

#### 渠道1: AI生成（推荐）

**优点**:
- 成本低
- 风格统一
- 可批量生成

**使用AI工具**:
1. **Midjourney**
   - 提示词模板见下文
   - 每张约$0.1-0.2

2. **Stable Diffusion**
   - 免费本地部署
   - 使用LoRA模型

3. **DALL-E 3**
   - 质量高
   - 理解能力强

**示例提示词（维吾尔族）**:
```
Uyghur ethnic style portrait, young woman wearing traditional Xinjiang silk dress (Adelais silk),
red and gold color scheme, ethnic jewelry, traditional flower hat (doppa),
intricate embroidery patterns, professional photography, high quality, detailed,
natural lighting, studio background --ar 1:1 --v 6.0
```

**批量生成**:
- 使用Midjourney的 `--repeat 4` 参数
- 每种民族生成50张需要约5-10美元

---

#### 渠道2: 在线图片库

**推荐网站**:
1. **Unsplash**: https://unsplash.com/
   - 搜索关键词：`Xinjiang ethnic`, `Uyghur`, `Kazakh`
   - 免费商用

2. **Pexels**: https://www.pexels.com/
   - 搜索关键词：`China ethnic`, `Xinjiang culture`
   - 免费高清

3. **Pixabay**: https://pixabay.com/
   - 搜索关键词：`ethnic group`, `traditional costume`

**注意事项**:
- 确认商用许可
- 注意版权
- 优先选择标注"Free for commercial use"

---

#### 渠道3: 自制/合作

**方式**:
1. 邀请新疆本地摄影师合作
2. 参加民族文化节拍照
3. 与民族艺术院校合作

**优点**:
- 真实、高质量
- 支持本地文化
- 可做推广合作

## 三、生成示例图制作

### 📊 生成示例图说明

每组示例图包含：
- **原图**: 普通人像
- **生成图**: 民族风格AI生成效果

### 🎯 目标

让用户直观看到：
- "上传这张照片后，会生成这样的效果"

### 📐 规范

| 项目 | 要求 |
|------|------|
| 格式 | JPG/PNG |
| 尺寸 | 512×512px |
| 大小 | < 300KB/张 |
| 对比 | 清晰对比原图和生成图 |

---

### 📸 示例图制作步骤

#### 步骤1: 准备原图

选择5张不同类型的原图：
- 男性青年照（1张）
- 女性青年照（1张）
- 儿童照（1张）
- 中年人照（1张）
- 老人照（1张）

**要求**:
- 正面照
- 光线均匀
- 背景简单
- 表情自然

---

#### 步骤2: 使用AI生成

对每张原图，生成5种民族风格：

**示例**:
```
原图: man_portrait.jpg

生成:
- uyghur_man.jpg (维吾尔族风格)
- kazakh_man.jpg (哈萨克族风格)
- mongol_man.jpg (蒙古族风格)
- kirgiz_man.jpg (柯尔克孜族风格)
- tajik_man.jpg (塔吉克族风格)
```

---

#### 步骤3: 制作对比图

使用设计工具制作：

**工具**:
- Photoshop
- Figma
- Canva
- 在线工具：https://www.canva.com/

**布局**:
```
+-------------------+-------------------+
|                   |                   |
|      原图         |      生成图        |
|  (普通照片)       |  (民族风格)        |
|                   |                   |
+-------------------+-------------------+
```

**标注**:
- 原图下方：标注"上传前"
- 生成图下方：标注"生成后 - 维吾尔族风格"

---

## 四、AI提示词模板

### 🎨 基础提示词结构

```
[民族类型] ethnic style, [年龄] [性别], [服装描述],
[图案元素], [配饰], [配色], [场景],
[质量要求]
```

---

### 📝 各民族提示词模板

#### 维吾尔族

**男性**:
```
Uyghur ethnic style, [age] [gender] portrait,
wearing traditional Xinjiang Adelais silk dress,
intricate embroidery patterns in red and gold,
traditional flower hat (doppa), ethnic jewelry,
high quality, detailed, professional photography,
natural lighting, studio background, portrait style
```

**女性**:
```
Uyghur ethnic style, young woman portrait,
wearing beautiful Adelais silk dress with traditional patterns,
elegant flower hat (doppa) with embroidery,
red and gold color scheme, ethnic jewelry,
detailed features, soft lighting, studio portrait,
high quality, professional photography
```

---

**通用**:

```
Kazakh ethnic style, [age] [gender] portrait,
wearing traditional Kazakh clothing with embroidery,
sheepskin coat elements, geometric patterns in blue and white,
ethnic jewelry and accessories,
grassland background, nomadic culture elements,
high quality, detailed, professional photography
```

## 📞 获取帮助

### 遇到问题？

1. **AI生成效果不好**:
   - 调整提示词
   - 检查参考图片质量
   - 尝试不同AI服务商

2. **生成速度慢**:
   - 使用豆包AI（最快）
   - 优化提示词长度
   - 检查网络连接

3. **民族特征不明显**:
   - 增加参考图片
   - 调整提示词权重
   - 添加更多民族元素
